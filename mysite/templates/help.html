Home

When you navigate to QuickShift you will arrive at the home screen. There, you will be greeted by 10 buttons, the top five serve as portals to the given restaurant's hub. To access these links you will need to sign in and have a manager account. Below, are the links for non-manager staff to view the current schedule.


Restaurant Manager Hub

Upon clicking one of the top five links on the home page, you will be able to access the Restaurant Manager Hub. If you are not logged in already you will be prompted to log in. Once that is completed, you will be greeted with an array of buttons and links.

The Home Button

This button will send you back to the home page.

The Log Out Button

This button will log you out and send you back to the homepage.

The Upload Button

This button will open a new tab titled "upload." In this tab, you will be able to upload the current schedule and email it. To begin, select the choose file button and choose the file you wish to upload. Once you have selected the file, click open. Note: this file must end in .csv or .xlxs. Once you are done click Upload Schedule, do not click this if no file is uploaded. If you uploaded the file correctly, you should be greeted with a "File Uploaded Successfully." Below, you will see 5 buttons corresponding to each restaurant, upon clicking any of these buttons it will send the schedule to all the staff and return you to Home.

The Schedule Button

This button will send you back to a public link in which anyone can view the current uploaded schedule.

The Availability Link

This link will send you back to a Google Sheet database that stores all staffs availability, email, and eligible positions. The first column will be everybody's email address. If there is no email address for a staff member, fill their box with "a@b.com." The second column houses everybodyâ€™s names. The next 14 columns hold everybody's availability (alternating between AM and PM). If that person is not available, put "x". If they are available to the server, put "s". If they are available to bartend, put "b". If they are available to host, put "h". If they are available to expo, put "e". The last column located at R2 holds the message that will be sent in the email containing the schedule. If you wish to type enter type "\n".

The Requests Off link

Upon clicking this link you will be redirected to Google Sheets, where you can view all requests. The Algorithm will override requests if there is not sufficient staff available. After generating the schedule, remember to delete all old requests.

The Needed Staff link

Upon clicking this link you will be redirected to Google Sheets, where you can view all the amount of needed staff for each position.

The Request Off Form Button

Upon clicking this link you will be redirected to Google Form, where staff can request off for a shift. You can share this link with all staff as well as create a QR code for in person at the restaurant. No sign-in or account is necessary for accessing this link.

The Generate Button

With a click of this button, the schedule will be generated and displayed below. Be patient because it may take a moment.

The Download Button

By clicking this link you can download the file as a .csv file.
